功能：
1. 端口扫描。可以指定IP范围和端口范围，进行端口扫描
2. IP扫描。可以扫描指定网段的存货IP
3. 网页扫描。内有三个功能
	1. 子域名扫描
	2. 目录扫描
	3. 指纹识别
4. 加解密。内有五个功能
	1. 栅栏加解密。可以指定每组的个数进行加密，以及栅栏爆破解密(把所有可能性全部列出)
	2. 凯撒加解密
	3. base64加解密
	4. MD5加密
	5. sha加密。 包含了sha的全部加密类型
5. 泛洪攻击模块。包含5个功能
	1. ARP欺骗。 将自己的MAC地址伪装成网关的MAC，使受害主机的流量发往自己的主机
	2. ARP攻击。 给定受害主机一个不存在的MAC地址，使其无法找到网关，导致网络中断
	3. ARP攻击网关。 不断向网关请求新的地址，使网关的资源池占满，让新加入的主机无法获得IP地址
	4. SYN半连接攻击。 向目标服务器不断发送SYN半连接的请求，使目标服务器资源枯竭
	5. SYN全连接攻击。 向目标服务器不断发送SYN全连接的请求，使目标服务器资源枯竭
6. 检测。包含5个功能
	1. MySQL爆破。
	2. Redis爆破
	3. SQL注入检测
	4. XSS注入检测
	5. Xray检测推送
7. 代理池。可以启动一个不断刷新的代理池，可以不断的获取新的代理IP
